# Homes & Beyond Property Mini Listings

Homes & Beyond is a Vue.js single-page prototype for a Cape Town short-term rental startup. It displays local property data in responsive cards and lets users search listings, sort by price, see unavailable properties, and mark favourites.

## Features

- Dynamic property cards from local data
- Search by property title or location
- Price sorting from low to high or high to low
- Not Available badge for unavailable listings
- Favourite/save button and saved listing count
- Responsive layout for desktop and mobile

## Project Setup

Install dependencies:

```sh
npm install
```

Run the development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```
